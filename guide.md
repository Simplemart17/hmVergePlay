# Project Overview
Create a cross-platform IPTV player app using the React Native Ignite template that efficiently supports both Live and VoD (Video on Demand) streaming. The app should be compatible with iPad, iPhone, Android phones, and Android tablets. Focus on delivering a smooth user experience with minimal buffering, utilizing M3U playlists and Xtream Codes links for streaming.

# App Requirements

## Features:
1. **User Authentication**:
   - Option for users to log in using Xtream Codes credentials.
   - Guest viewing option without authentication.

2. **M3U & Xtream Code Support**:
   - Ability to load and play channels from M3U playlists.
   - Support for streaming through Xtream Codes API.

3. **Channel Guide**:
   - Display a user-friendly EPG (Electronic Program Guide) for live channels.
   - Option to browse channels by categories.

4. **Player Features**:
   - Full-screen video playback.
   - Options for playback controls (play, pause, rewind, forward).
   - Picture-in-Picture (PiP) support for multitasking.
   - Smooth streaming with minimal buffering using adaptive bitrate streaming techniques.

5. **Design and User Experience**:
   - Implement a modern, visually appealing UI that is intuitive to use.
   - Provide theming options for light and dark mode.
   - Ensure responsive design to adapt seamlessly across all devices (iPads, iPhones, Android phones, and tablets).

6. **Offline Capability**:
   - Option to download content for offline viewing.

## Technical Requirements:
- Use the latest version of React Native with Ignite CLI.
- Ensure compatibility with both iOS and Android platforms.
- Implement state management using MobX-State-Tree (MST).
- Use React Navigation for seamless navigation between screens.
- Utilize React Native Video or similar library for video playback.
- Follow best practices for performance optimization to ensure smooth streaming experience.
